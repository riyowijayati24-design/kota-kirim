Deskripsi

Kota Kirim adalah aplikasi sederhana untuk mendukung proses pengiriman barang antar kota.
Aplikasi ini menyediakan fitur:

Input data pengirim & penerima

Perhitungan estimasi biaya kirim

Pelacakan status pengiriman

Penyimpanan riwayat paket

Proyek ini dibuat untuk memenuhi tugas pemrograman dan dokumentasi menggunakan GitHub, markdown, serta version control.
